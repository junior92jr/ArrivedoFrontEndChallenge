# ArrivedoFrontEndChallenge

## Dependencies
- npm
- bower

## Instructions
1) git clone https://github.com/junior92jr/ArrivedoFrontEndChallenge.git
2) cd ArrivedoFrontEndChallenge/
3) npm install
4) bower install
5) gulp connect
6) http://127.0.0.1:8888/
7) You can try now with the app interaction
